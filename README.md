# 🧠 AI Maze Solver with Q-learning

本專案使用 Q-learning 強化學習方法，訓練智能代理人從起點學習走出迷宮。  
適合作為 AI 初學者的強化學習入門實作！

---

## 📌 專案目標

- 模擬一個 2D 迷宮環境
- 定義強化學習中的狀態、動作、獎勵
- 使用 Q-learning 訓練 agent 學會從起點到終點
- 展示學習結果與最佳路徑

---

## 🧠 使用技術

| 技術 | 說明 |
|------|------|
| Python | 實作主要邏輯 |
| NumPy | Q-table 儲存與矩陣計算 |
| Q-learning | 強化學習演算法，用於決策優化 |
| ε-greedy 策略 | 平衡探索與利用 |
